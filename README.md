# calculadora-herrera
¿Para qué sirve esta calculadora? Ingresa los datos de un producto nuevo y la herramienta te calcula el precio de venta recomendado — sin IVA y con IVA — basado en la familia del producto. También revisa automáticamente si hay competencia con un precio más bajo para que puedas tomar mejores decisiones antes de confirmar una compra.
